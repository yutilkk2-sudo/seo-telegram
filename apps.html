源代码

目前，我们专注于开源，让开发者能够使用我们的 API 快速构建一些东西。我们已经发布了 Android、iOS、Web 和桌面应用程序（Win、macOS 和 Linux）以及 Telegram 数据库库的代码。

该代码允许安全研究人员全面评估我们的端到端加密实现。还可以独立验证Google Play 和 App Store 上提供的 Telegram 应用程序是否是使用我们在 GitHub 上发布的相同代码构建的。
